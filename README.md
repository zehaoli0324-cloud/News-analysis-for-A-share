================================================================
【部署到 Render（免费公网访问）】
================================================================

--- 第一步：上传到 GitHub ---

在项目文件夹里执行以下命令：

  git init
  git add .
  git commit -m "A股量化分析平台 V2"
  git branch -M main
  git remote add origin https://github.com/你的用户名/仓库名.git
  git push -u origin main

--- 第二步：在 Render 创建服务 ---

1. 访问 https://render.com，注册并登录
2. 点击 New → Web Service
3. 选择你的 GitHub 仓库，点击 Connect
4. 按以下配置填写：

   Name          : 随意，例如 astock-platform
   Region        : Singapore（国内访问延迟最低）
   Branch        : main
   Runtime       : Python 3
   Build Command : pip install -r requirements.txt
   Start Command : gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --threads 8 --timeout 600
   Instance Type : Free

5. 点击 Create Web Service，等待约3-5分钟部署完成

部署成功后 Render 会给你一个公网地址，格式如：
https://astock-platform-xxxx.onrender.com

--- 注意事项 ---

免费版会休眠：15分钟没有访问后服务器会休眠，
              下次访问时需要等约20秒唤醒，唤醒后完全正常。
              分析过程中不会休眠，不影响使用。

如需7×24小时不休眠，升级为 Starter 方案（约$7/月）即可。

================================================================
【API Key 获取方式】
================================================================

功能1 & 功能2（资金选股/单股分析）需要 Tushare Token：
  注册地址：https://tushare.pro/register
  注册后在个人中心复制 Token，填入页面即可

功能3（新闻AI分析）需要硅基流动 API Key：
  注册地址：https://cloud.siliconflow.cn
  新用户注册即送14元免费额度
  使用模型：DeepSeek-V3（免费版每日100次）
  每次完整分析费用：约0.3元

所有 Key 均在页面上实时输入，不保存在代码里，不会泄露。

================================================================
【功能说明】
================================================================

--- 功能三：新闻资讯AI深度分析 ---

输入股票代码 + 硅基流动 Key，点击开始，系统会：

  1. AI扩展关键词（竞争对手、产业链、热词）
  2. 并发采集7类数据（个股新闻/板块/宏观/货币/行情/龙虎榜/大股东）
  3. GNews多维搜索（最多50条查询，覆盖公司/产业链/政策/竞品）
  4. DeepSeek-V3 生成投资简报（约5-8分钟）

报告风格：交易员视角，段落叙述，有倾向性判断。
报告包含九个章节：核心结论/事件复盘/原因分析/基本面估值/
                  资金解读/多空分歧/交易员判断/投资建议/核查清单

分析完成后可下载：
  - PDF格式投资简报（含新闻附录）
  - TXT格式全部新闻原文

================================================================
【常见问题】
================================================================

Q: PDF下载后是TXT格式？
A: 你的电脑缺少中文字体，系统自动回退为HTML格式。
   Windows用户确认 C:\Windows\Fonts\msyh.ttc 存在即可。
   Render部署版本会自动安装字体，不会有这个问题。

Q: 分析报告被截断，没有写完？
A: DeepSeek免费版偶尔会提前截断输出。可以稍后重新分析，
   或在"继续追问AI"对话框里输入"请继续写第X章"。

Q: GNews搜索结果很少？
A: 部分行业在中文互联网的覆盖度有限属正常现象。
   核心数据来自东方财富和龙虎榜，GNews是补充来源。

Q: 提示 SiliconFlow API 错误？
A: 检查 API Key 是否正确，以及账户余额是否充足（至少0.5元）。
   硅基流动免费额度用完后需要充值才能继续使用。

Q: Tushare 接口报权限错误？
A: 部分接口需要 Tushare 积分。注册后每日签到可积累积分，
   或直接充值开通相应权限。
